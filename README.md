# CodeAlpha Language Translation Tool

## Description

A Python-based Language Translation Tool developed as part of the CodeAlpha Artificial Intelligence Internship.

The application allows users to enter text, select source and target languages, and instantly translate text using the Google Translation API.

## Features

* User-friendly GUI using Tkinter
* Supports multiple languages
* Source and target language selection
* Real-time translation
* Copy translated text button
* Error handling

## Technologies Used

* Python
* Tkinter
* Deep Translator API

## Installation

1. Clone the repository

2. Install dependencies

pip install -r requirements.txt

3. Run the project

python main.py
